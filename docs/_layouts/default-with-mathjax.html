This site presents demos which supplement the article, “Six Ways to Implement Natural Language Processing in the Social Sector.” (replace link once published) Please refer to the article for an overview of NLP, its foundational techniques, and more on applications in the social sector.

The demos for each technique were primarily built on a dataset of 28,000 bills signed over the past ten years from five states (California, New York, South Dakota, New Hampshire, and Pennsylvania). All supporting code can be found in Github.
